# DivZero Site

Static stakeholder preview site for DivZero.

## Files

- `index.html` - page markup and metadata.
- `styles.css` - visual styling.
- `script.js` - typewriter, form, reveal, and event accordion behavior.
- `2026-opening.jpg`, `og.png`, `apple-touch-icon.png` - published image assets.

## Preview

Open `index.html` directly in a browser, or use any static file server from this directory.

## Publish

This repo is ready for GitHub Pages using the `main` branch root directory.
